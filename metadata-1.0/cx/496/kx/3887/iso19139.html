Abstract:

This polygon shapefile contains boundaries of provinces in Equatorial Guinea as of January 2007. These are the level 1 administrative units (ADM1) for Equatorial Guinea. This layer is part of the Map Library dataset containing line and polygon boundaries for individual countries on the African continent.

Purpose:

The Map Library is a project of the UK charity “The Map Maker Trust." It aims to make public domain mapping data more readily available for development in all kinds of fields, from health care to urban management, from de-mining to environmental protection.

Data Quality Information:

Lineage:

Statement:

The boundary status and administrative names have been verified against various sources of information. The information given by: www.wikipedia.org and www.statoids.com and 'SALB' proved to be most reliable. All often had links to governmental web-sites or other reliable references to verify it against. The actual polygon set has become an amalgamation of other datasets available in the public domain. Great care has been taken in 'edge matching' each polygon and we believe therefore the 'topology' is as good as one can get it. Where two different sources for the polygon set were available, or where a different polygon set of two neighbouring nations were used, in this amalgamation process the most detailed line work was used. The following web sites were used to amalgamate and update the polygon set: http://igskmncnwb015.cr.usgs.gov/adds/geolist.php (FEW-NET, Africa Data Dissemination Service) http://www.fao.org/geonetwork/srv/en/main.home http://www.diva-gis.org http://www3.who.int/whosis/gis/salb/salb_MDATA.htm http://www.depha.org/ http://www.sahims.net/ All countries have been updated to at least ADM2 level or to a next level where reliable information was found. Where in the old dataset more divisions were present, the polygon boundaries and names were left in place. All updated names in the database are in Upper &amp; Lower case. In general, one can assume that any names in the database solely in capitals reflect old unverified or possibly obsolete administrative division information (i.e. boundary and names).
