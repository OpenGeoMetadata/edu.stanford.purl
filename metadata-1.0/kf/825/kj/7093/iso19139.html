Abstract:

This line shapefile represents the federally owned land features (for example, national parkways and wild and scenic rivers) of the United States. No data exists for Hawaii. This layer is part of the 2014 ESRI Data and Maps collection for ArcGIS 10.2.

Purpose:

U.S. National Atlas Federal Land Lines provides the linear federally owned land features for geographic display and analysis at regional and national levels.

Process Step:

This file was originally digitized by the National Mapping Division based on the sectional maps contained in 'The National Atlas of the United States of America®' published by the USGS in 1970. The sectional maps were updated during 1978-1981 and digitized in the early 1980's. The data were updated in 1995 using 1:1,000,000-scale and 1:2,000,000-scale Bureau of Land Management State base maps. These data were published on CD-ROM in 1995. Using ARC/INFO® software, the DLG optional format files were converted to ARC/INFO® coverages using the DLGARC command. Only linear federal land features and attribute information were extracted for inclusion. The individual State coverages were then merged together using the ARC/INFO® command APPEND. The resulting data set was ATLAS-FEDL (Federal Land Features of the U.S. (linear)). Using ATLAS-FEDL the spatial and attribute data were updated using the Wild and Scenic Rivers website. The spatial data was aligned with ATLAS-HY (Hydrography Features of the U.S.) and ATLAS-RD (Major roads and trails of the U.S.). Sudbury, Assabet, and Concord Wild and Scenic Rivers, MA were added using Linear Federal Land Features of the United States 1999, Wild and Scenic Rivers website, and ATLAS-HY (Hydrography Features of the U.S.) in 2000. The resulting data set was Federal Land Features of the United States - Parkways and Scenic Rivers 2000. Wilson Creek Wild and Scenic River, NC was added and Baltimore-Washington Parkway, MD and Suitland Parkway, MD were added using Federal Land Features of the United States - Parkways and Scenic Rivers 2000, Wild and Scenic Rivers website, National Park Guide website, USGS Geographic Data website, ATLAS-HY (Hydrography Features of the U.S.), ATLAS-RD (Major roads and trails of the U.S.) in 2001.

Date And Time: 2001-01-01T00:00:00

Removed unneeded attributes. Created ArcGIS® layer file (.lyr), projection file (.prj), and spatial indices. Converted the data set to SDC.
