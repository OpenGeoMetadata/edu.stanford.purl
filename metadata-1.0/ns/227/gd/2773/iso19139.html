Abstract:

This line shapefile represents bays and estuaries in California. This file is derived from the generalized county coastline file. This layer is part of a collection of data created by the California Department of Fish & Game.

Purpose:

These data are intended for researchers, students, policy makers, and the general public for reference and mapping purposes, and may be used for basic applications such as viewing, querying, and map output production.
