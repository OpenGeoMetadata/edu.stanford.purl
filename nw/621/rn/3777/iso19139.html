Abstract:

This line shapefile shows contour coverage of Monterey Bay at 100-meter intervals. These data were created from the 20-Meter Bathymetry Grid (bathy20m) using the spatial analyst. Shapefiles for this layer are available in WGS84 and UTM Zone 10 (NAD83) projections and in Generic Mapping Tools (GMT) format. This layer is part of the GIS Data of the Monterey Bay collection, a compilation of data and imagery of the Monterey Bay area, including coastline, imagery, and bathymetry.

Purpose:

These data were created as a vector representation of a raster-based bathymetry surface. This collection of data provides documented layers of of the Monterey Bay to persons/institutions of interest throughout the research and educational communities.

Supplemental Information:

All data are registered to the WGS84 datum with two versions of each feature, image, and grid coverage included in the collection. One is in a Geographic (decimal degrees) coordinate system and the second is in a Universal Transverse Mercator (UTM) zone 10 projection. A GMT version of this layer is also available. For strict accuracy and hard-copy production requiring feature, grid, and/or image data, the UTM projection coverages should be used. In fact, some ArcView functions will not be available unless the data are displayed in a projection. This is because a Geographic Coordinate System is NOT a projection but rather a spherical coordinate system dealing directly in latitude and longitude. However, at the scale of maps covering Monterey Bay, the errors produced by ignoring this fact are small.

Process Step:

The grid bathy20m was run through filter four times to produce a smoothed grid from which the contour coverage was created in ArcView with the spatial analyst. The coverage was then created from the shapefile with the ARC/INFO command [shapearc] then the coverage was run through ARC/INFO's clean function to reduce the size, ensure proper topology and sort the data. Then it was converted back to a shapefile with arcshape and all the data fields except shape and contour were removed from the data table. The shape field was then indexed to increase draw speed. ARC/INFO commands used: filter bathy20m bathy20m_f4 low 4 export grid bathy20m_f4 bathy20m_f4 (import the grid to ArcView and create the contour shapefiles with spatial analyst) shapearc 100m_contours 100m_contours clean 100M_CONTOURS # # 0.0002 line arcshape
