Abstract:

This polygon shapefile depicts California halibut trawl grounds as defined by California Fish and Game Code 8495 and California Code of Regulations, Title 14, Section 124. These data define areas open and closed to trawling within the state waters of California with a valid California Halibut Trawl Permit. This data was modified in October 2013 to add full grounds to the file since the previous version only included sub-areas.

Purpose:

This coverage provides cartographic representation of the halibut trawl grounds of Southern California.
