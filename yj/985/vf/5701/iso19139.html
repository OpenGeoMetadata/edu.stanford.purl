Abstract:

This point shapefile contains point locations for zip codes in United States from July until September, 2006. Attributes included in the data are polygon (area boundary ID), area/perimeter, county names, FIPS and USPS codes, and alternate names. These data are updated quarterly using the latest feeds from the U.S. Postal Service with additional sources being used to cross-check the data.

Purpose:

This layer provides zip code data for view and analysis in GIS and interactive web mapping applications.
