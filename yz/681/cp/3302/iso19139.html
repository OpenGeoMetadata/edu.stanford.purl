Abstract:

This point shapefile represents the populations of the U.S. Census blocks as centroids for the United States. U.S. Census blocks nest within all other tabulated census geographic entities and are the basis for all tabulated data. This layer is part of the 2014 ESRI Data and Maps collection for ArcGIS 10.2.

Purpose:

U.S. Census Block Centroid Populations provides population and other demographic information for each U.S. Census block centroid within the United States.

Process Step:

The following steps were performed by Esri: Created the block centroids from the 2010 Census Redistricting Data (Public Law 94-171) Summary File. Reordered and renamed the fields. Added fields STATE_FIPS, CNTY_FIPS, STCOFIPS, TRACT, and BLOCK and created their values from parsing the values of the FIPS field. Projected the data set to World Geodetic System of 1984 (WGS 1984). Converted the data set to SDC. Created ArcGIS® layer file (.lyr), projection file (.prj), and spatial indices.
